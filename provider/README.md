# dubbo-java-provider-demo

Java Dubbo provider demo: register a proto-based Triple service to Nacos by interface.

## What it contains

- `src/main/proto/helloworld/HelloWorld.proto`: protobuf service and message definition.
- `DubboHelloWorldServiceTriple.HelloWorldServiceImplBase`: generated by `dubbo-maven-plugin`.
- `HelloWorldProvider`: `@DubboService` provider implementation.
- `application.properties`: Nacos registry, `tri` protocol, and interface-level registration.

## Generate and compile

```powershell
mvn compile
```

The Dubbo Maven plugin writes generated Java sources to `src/main/java`.

## Run

```powershell
$env:NACOS_SERVER_ADDR = "10.80.1.67:18848"
$env:NACOS_NAMESPACE = "public"
$env:NACOS_USERNAME = "nacos"
$env:NACOS_PASSWORD = "your-password"
mvn spring-boot:run
```

Registered interface:

```text
cn.mobile.ivy.demo.helloworld.HelloWorldService
```

Triple path:

```text
/cn.mobile.ivy.demo.helloworld.HelloWorldService/sayHello
```
